Found a bug? Have a feature idea?

For bugs, the debug log is the fastest way to diagnose what went wrong — see the guide below.

How to capture the debug log
  1. 01
    Load the problematic file Go to the main page and drop the file that's causing issues.
  2. 02
    Open the debug log Scroll to the bottom of the page and click Debug log to expand the panel.
  3. 03
    Copy all entries Click the Copy log button in the top-right of the debug panel. All timestamped entries are copied to your clipboard.
  4. 04
    Paste it below Fill out the form and paste the log into the Debug log field.
